# **App Name**: MindfulJourney

## Core Features:

- AI Chat Assistant: Real-time mental wellness conversation powered by Gemini API with context-aware responses and an empathetic tone.
- Mood Tracker: Daily mood input with emoji scale and optional notes. Gemini analyzes weekly patterns for insights.
- Smart Journal: A text based Journal where users can log journal entries with features like summarizing, sentiment detection, and personalized wellness tip recommendations powered by Gemini API.
- Guided Exercises: Provides UI-driven breathing exercises, mindfulness tips, sleep relaxation techniques, and daily gratitude prompts to the user.
- User Authentication: Secure Firebase authentication (Email/Password and Google login) for user data protection.
- Data Storage: Utilize Firestore to persist realtime data, including mood logs, journal entries, and user settings, allowing users to retain this data between sessions.
- Insight Tool: The AI tool can reason on information extracted from a journal, as well as logged mood data to generate summaries.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for calmness and serenity.
- Background color: Very light lavender (#F5F5FF) for a peaceful, uncluttered feel.
- Accent color: Mint green (#98FF98) to represent growth and tranquility.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its modern and neutral aesthetic.
- Use simple, line-based icons to maintain a minimalist design. Ensure icons are intuitive and align with the theme.
- Employ a clean, card-based dashboard layout to present information clearly. Prioritize mobile-first responsive design.
- Incorporate subtle animations for user interactions and transitions. The breathing exercise page uses animations.
